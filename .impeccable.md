# .impeccable.md

## Design Context

### Users
20~40대 Toss 사용자 중 홈트레이닝에 관심 있지만 PT 비용이 부담스러운 직장인과 주부. 그들은 "내가 제대로 하고 있는지" 즉시 확인하는 실시간 피드백을 가장 원하며, 유튜브 영상보다는 자신의 체형과 능력에 맞춘 개인화된 운동을 추구합니다. 월 12,900원으로 PT의 1/40 가격대에서 체형 관리의 작은 성공을 경험하려는 사람들입니다.

### Brand Personality
**효율적이고 신뢰할 수 있으며 친근한** 피트니스 동료. Toss의 투명성과 단순함을 이어받아, AI 기술은 검증된 과학에 기반하되 어렵지 않게 전달합니다. 사용자의 작은 진전도 솔직하게 축하하고, 실패는 개선 기회로 재해석하는 따뜻한 톤입니다.

### Aesthetic Direction
TDS (Toss Design System) 준수. @toss/tds-mobile 컴포넌트 전용. 커스텀 미적 방향 없음.

### UX Principles
- **실시간 응답성**: 자세 인식과 음성 피드백은 500ms 이내로 반응하여 운동 몰입도를 해치지 않음
- **진입 장벽 최소화**: 카메라만으로 시작, 복잡한 설정 없이 3탭 안에 운동 시작
- **시각적 확신**: 관절 각도, 진행도, 칼로리 소모를 명확한 수치와 그래프로 표시하여 사용자가 "하고 있다"는 확신을 가지게 함
- **점진적 도전**: 초보자는 기본 3개 운동에서 시작하되, 진도에 맞춰 새로운 운동과 커스터마이징 옵션이 자연스럽게 열림
- **프라이버시 신뢰**: 카메라 데이터 처리 방식과 저장 정책을 명확히 전달하여 신체 정보 활용에 대한 우려 해소
